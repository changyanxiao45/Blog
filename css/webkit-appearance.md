# -webkit-appearance
## 概述
-webkit-appearance 是一个 不规范的属性（unsupported WebKit property），它没有出现在 CSS 规范草案中。
改变按钮和其他控件的外观，使其类似于原生控件。
## 语法
-webkit-appearance：none | button | button-bevel ....
